# krasotkaDB

**krasotkaDB** — это десктопное приложение для управления базой данных салона красоты «Красотка». Программа предназначена для автоматизации рабочих процессов администратора: ведения учета клиентов, управления расписанием мастеров, а также каталогизации оказываемых услуг.

## 🛠 Технологии

* **Язык программирования:** C#
* **Пользовательский интерфейс:** WPF (Windows Presentation Foundation)
* **Архитектура:** Классическое многооконное приложение
* **База данных:** SQL (скрипты для развертывания включены в проект)

## 🗂 Основной функционал (Структура проекта)

Интерфейс приложения разделен на несколько ключевых модулей:

* **👩‍🦰 Клиенты (`Clients.xaml`):** Управление клиентской базой (добавление, редактирование, просмотр истории).
* **💅 Мастера (`Masters.xaml`):** Учет сотрудников салона и их специализаций.
* **📋 Услуги и категории (`Services.xaml`, `ServTypes.xaml`):** Ведение прайс-листа и справочника предоставляемых услуг с разбивкой по типам.
* **📅 Ежедневный учет (`EveryDay.xaml`):** Модуль для работы с ежедневным расписанием, записями клиентов к мастерам.
* **🔌 Управление БД (`OpenDB.xaml`):** Модуль для настройки и установки соединения с базой данных.

## 🚀 Установка и запуск

### 1. Подготовка базы данных
В корне проекта находятся готовые SQL-скрипты для быстрого развертывания БД:
1. Выполните скрипт `create tables.sql` в вашей СУБД для создания структуры таблиц.
2. Для заполнения базы начальными (тестовыми) данными выполните скрипт `insert tables.sql`.
3. Файл `select tables.sql` содержит готовые запросы, которые могут пригодиться для проверки данных.

### 2. Запуск приложения
1. Склонируйте репозиторий на свой компьютер.
2. Откройте файл решения `krasotkaDBLevonTEST.sln` в **Microsoft Visual Studio**.
3. Убедитесь, что установлены необходимые компоненты для разработки классических приложений .NET (WPF).
4. Настройте строку подключения к вашей базе данных (при необходимости, через интерфейс `OpenDB` или конфигурационные файлы).
5. Нажмите `F5` или выберите `Сборка -> Запуск` для старта приложения.
